## еӣӣгҖҒй…ҚзҪ®зі»з»ҹ

### и®ҫи®ЎеҺҹеҲҷ
- **YAML жҳҜеҚ•дёҖзңҹзӣёжқҘжәҗ**пјҲ`config/config.yaml`пјү
- **JSON жҳҜ Dashboard UI иҰҶзӣ–**пјҲ`data/system_config.json`пјү
- **зҺҜеўғеҸҳйҮҸжңҖй«ҳдјҳе…Ҳзә§**пјҲ`.env` / shell environmentпјү
- **ж— еҶ—дҪҷеҗҢжӯҘ**пјҡDashboard зј–иҫ‘еҗҺйҖҡиҝҮ `Config._merge_system_config()` иҮӘеҠЁеҗҲе№¶

### й…ҚзҪ®еҗҲе№¶йЎәеәҸ
```
config/config.yaml < data/system_config.json < .env / shell environment
```

### е…ій”®й…ҚзҪ®ж–Үд»¶
| ж–Үд»¶ | з”ЁйҖ” |
|------|------|
| `config/config.yaml` | дё»й…ҚзҪ®пјҲеҗ«жүҖжңүиҝҗиЎҢж—¶й»ҳи®ӨеҖјпјү |
| `data/system_config.json` | Dashboard UI жҢҒд№…еҢ–пјҲж•Ҹж„ҹдҝЎжҒҜеңЁжӯӨпјү |
| `.env` | зҺҜеўғеҸҳйҮҸпјҲAPI еҜҶй’ҘгҖҒеҮӯиҜҒзӯүпјү |
